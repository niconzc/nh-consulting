# Sales-Strategie — NH Consulting

**Fokus 2025:** Erste 5–8 Kunden über Empfehlungen und direkten Outreach

---

## Strategie-Überblick

Als nebenberuflich Startender ohne Portfolio ist das Ziel:
1. **Schnell** erste Kunden gewinnen (Referenzen > Umsatz anfangs)
2. **Selektiv** vorgehen (lieber 1 gutes Projekt als 5 schlechte)
3. **Netzwerk** nutzen (Cloudunify, persönliches Umfeld)

---

## Kanal 1: Persönliches Netzwerk (Priorität 1)

### Vorgehen

**Schritt 1:** Liste aller Bekannten, Kollegen, Ex-Kollegen, die:
- ein eigenes Unternehmen haben
- eine schlechte/veraltete Website haben
- jemanden kennen, der gerade eine Website sucht

**Schritt 2:** Persönliche WhatsApp/Telegram-Nachricht (nicht LinkedIn-kalt):

```
Hey [Name],

ich hab gerade nebenberuflich NH Consulting gestartet –
ich mach moderne Websites und AI-Automatisierungen für KMU.

Weißt du zufällig jemanden in deinem Umfeld, der eine
neue Website braucht oder digitalisieren will?
Würde mich über eine Weiterleitung sehr freuen.

Danke und lg,
Nico
```

> **Warum das funktioniert:** Direkt, keine Scheu, um Hilfe bitten ist legitim.

**Schritt 3:** Bei Interesse: sofort Erstgespräch vorschlagen.

## Kanal 2: LinkedIn (Priorität 2)

### Profil-Optimierung

**Headline:**
```
Nicolas Hauser · IT-Consultant & Webentwickler | NH Consulting | 
Websites · AI-Integration · Automatisierung
```

**About-Sektion:**
```
Ich baue moderne Websites und integriere AI-Tools in bestehende
Geschäftsprozesse – für KMU in Österreich und Deutschland.

Was ich mache:
→ Professionelle Websites (Next.js, TypeScript)
→ AI-Chatbots und Automatisierungen
→ Interne Tools und API-Integrationen

Ich arbeite direkt mit Geschäftsführern und Inhabern –
ohne lange Agentur-Prozesse, mit fixen Preisen und klaren Deadlines.

📩 Erstgespräch: nh-consulting.at/kontakt
```

### Content-Strategie (1–2 Posts/Woche)

**Post-Formate die funktionieren:**

1. **"Vorher/Nachher"** (sobald erstes Projekt):
   ```
   Vorher: 2008-Webauftritt, nicht mobil, keine Anfragen
   Nachher: Moderne Website, +X% Anfragen in Monat 1
   [Screenshot]
   Was hat den Unterschied gemacht? [3 Punkte]
   ```

2. **"Was ich gelernt habe"** (aus Projekten, ohne Kundendaten):
   ```
   Fehler den ich bei Website-Projekten immer wieder sehe:
   
   Kunden kommen mit "ich brauche eine Website" –
   aber was sie wirklich brauchen ist [echtes Problem].
   
   3 Fragen, die ich vor jedem Projekt stelle: ...
   ```

3. **"Kurzer AI-Tipp"** (zeigt Expertise):
   ```
   In 10 Minuten: Wie du mit Claude einen automatischen
   E-Mail-Beantwortungsprozess für Kundenanfragen baust.
   
   [Schritt-für-Schritt]
   
   Brauchst du das für dein Unternehmen? DM me.
   ```

### Direkter LinkedIn-Outreach

**Wen ansprechen:** Geschäftsführer / Inhaber von KMU in Salzburg/Österreich, die:
- Veraltete Website erkennbar haben
- In IT-affinen Branchen sind (aber keine IT-Firmen)
- Wachstumsphasen sind (neue Mitarbeiter, Büro etc.)

**Nachricht-Vorlage:**
```
Betreff: Kurze Frage zu Ihrer Website

Hallo [Vorname],

ich bin Nicolas, Webentwickler aus Salzburg.

Ich habe Ihre Website gesehen – [echte, spezifische Beobachtung,
z.B. "toll, aber auf Mobile lädt sie sehr langsam"].

Ich helfe KMU, ihre Online-Präsenz zu modernisieren –
schnell, zu fixen Preisen, ohne Agentur-Umwege.

Hätten Sie Interesse an 20 Minuten Austausch?
Kein Verkaufsgespräch, nur schauen ob ich helfen kann.

Grüße
Nicolas Hauser
nh-consulting.at
```

**Regeln:**
- Nie copy-paste – immer personalisieren
- Max. 10 Nachrichten/Woche (LinkedIn-Limits + Qualität)
- Nachfassen nach 1 Woche wenn keine Antwort (einmal reicht)

## Kanal 3: Cloudunify-Netzwerk (Priorität 1)

Du bist Sales Manager – das ist ein Goldgriff.

**Vorgehen:**
- Kunden / Kontakte von Cloudunify, die du gut kennst: direkt fragen
- Intern: Kolleginnen und Kollegen informieren ("Wenn jemand eine Website braucht...")
- Ex-Kunden von Cloudunify, die jetzt woanders sind

> ⚠️ Prüfe Arbeitsvertrag auf Wettbewerbs- und Abwerbungsklauseln, bevor du Cloudunify-Kunden direkt ansprichst.

## Kanal 4: Empfehlungsprogramm (ab Monat 2)

Sobald 1–2 zufriedene Kunden:

```
"Jede erfolgreiche Empfehlung = 10% Rabatt auf Ihren nächsten Auftrag"
```

Einfach, direkt, effektiv. Kein kompliziertes System.

---

## Preisverhandlungen — Leitfaden

**Wenn jemand "zu teuer" sagt:**

Option 1 — Scope reduzieren:
```
"Kein Problem. Was wäre, wenn wir mit einer 4-Seiten-Website starten?
Das wäre für €1.199,– möglich. Wir können später erweitern."
```

Option 2 — Zahlungsplan:
```
"Ich verstehe das Budget-Thema. Wäre ein Zahlungsplan hilfreich?
Z.B. in 3 Monatsraten à €500?"
```

Option 3 — Wert erklären:
```
"Ich verstehe. Zum Vergleich: Eine Stunde schlechte Wirkung auf
potenzielle Kunden durch eine veraltete Website kostet mehr
als die einmalige Investition in eine professionelle."
```

**Was du NICHT tun solltest:**
- Preis ohne Gegenleistung reduzieren
- Unter €800 für eine Website (Qualität leidet, Kunde ist schwierig)
- Auf Erfolgshonorar oder % vom Umsatz eingehen (anfangs)
