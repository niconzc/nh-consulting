# Startup-Checkliste — NH Consulting

> **Sofort-Aktionen** für den Start. Abarbeiten in Reihenfolge.

---

## 🔴 WOCHE 1 — Pflicht (sofort)

### Rechtliches
- [ ] Handysignatur aktivieren (wichtig für usp.gv.at) → handy-signatur.at
- [ ] Gewerbe anmelden → usp.gv.at (IT-Dienstleistungen, freies Gewerbe)
- [ ] NeuFÖG-Formular miteinreichen (Stempelgebühren-Befreiung)
- [ ] FinanzOnline-Zugang erstellen → finanzonline.at
- [ ] Steuerliche Erfassung (Fragebogen) in FinanzOnline ausfüllen
- [ ] Hauptjob-Vertrag lesen: Gibt es eine Konkurrenzklausel oder Nebentätigkeitsverbot?

### Domain & Online-Präsenz
- [ ] nh-consulting.at registrieren (z.B. über world4you.at oder nic.at)
- [ ] GitHub-Profil professionalisieren (Bio, Location, Link)
- [ ] LinkedIn-Profil aktualisieren: „Inhaber NH Consulting" in Headline ergänzen

### Tools & Setup
- [ ] 1Password: Vault für "NH Consulting" erstellen
- [ ] Neue E-Mail: office@nh-consulting.at (Google Workspace ~€6/Monat oder Zoho Mail gratis)
- [ ] GitHub Repository `nh-consulting` erstellen und dieses Repo pushen

---

## 🟡 WOCHE 2 — Wichtig

### Finanzen
- [ ] Geschäftskonto eröffnen (Holvi-Empfehlung: holvi.com/at)
- [ ] IBAN des Geschäftskontos notieren für Rechnungsvorlage
- [ ] Erste Rücklage anlegen: Ziel €500 "Steuerpuffer"
- [ ] Excel/Spreadsheet für Einnahmen-Ausgaben starten (oder Holvi nutzen)

### Website vorbereiten
- [ ] Vercel-Account erstellen (kostenlos) → vercel.com
- [ ] GitHub Pages oder Vercel mit Domain nh-consulting.at verbinden
- [ ] Placeholder-Seite deployen ("Coming soon" oder einfache Landing Page)

### Erste Materialien
- [ ] Rechnungsvorlage aus diesem Repo ausfüllen (Adresse, IBAN)
- [ ] Angebotsvorlage erstellen
- [ ] Visitenkarte designen (optional, digital reicht anfangs)

---

## 🟢 MONAT 1 — Website & Erster Kunde

### Website
- [ ] Vollständige NH Consulting Website entwickeln und live schalten
- [ ] Kontaktformular mit echtem E-Mail-Versand (Resend oder Formspree)
- [ ] Impressum, Datenschutz, Cookie-Banner
- [ ] Google Analytics einrichten
- [ ] Google Search Console einrichten

### Ersten Kunden gewinnen
- [ ] 10 potenzielle Kunden aus bestehendem Netzwerk identifizieren
- [ ] LinkedIn-Nachricht formulieren (Vorlage in SALES_STRATEGY.md)
- [ ] 3 Erstkontakte pro Woche anschreiben
- [ ] Cloudunify-Kollegen / -Kontakte informieren

### Portfolio
- [ ] Mindestens 1 Demo-Projekt / eigenes Projekt als Referenz
- [ ] Screenshots / Screen-Recordings erstellen
- [ ] Portfolio-Seite vorbereiten (kann zuerst ein Projekt sein)

---

## 📅 MONAT 2–3 — Stabilisierung

- [ ] Steuerberater konsultieren (kostenlose Erstberatung WKO Salzburg nutzen)
- [ ] Ersten Wartungsvertrag abschließen (bei erstem Website-Kunden anbieten)
- [ ] Referenzschreiben vom ersten Kunden einholen
- [ ] LinkedIn-Posts starten (1x/Woche reicht anfangs)
- [ ] Prozesse dokumentieren: wie läuft ein Projekt ab?
